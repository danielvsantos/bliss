# Bliss Finance

## Open-Source Wealth Intelligence Platform

Bliss is a multi-currency personal wealth intelligence platform. It ingests bank transactions (Plaid or CSV), classifies them with a 4-tier AI pipeline, tracks investment portfolios with real-time pricing, and generates AI-powered financial insights.

## Architecture at a Glance

```
Browser (React SPA) → Nginx (:8080)
  ├→ Next.js API (:3000)  — Auth, REST, Prisma ORM
  └→ Express Backend (:3001) — BullMQ workers, AI pipelines, Portfolio engine
  Both services ↓
  PostgreSQL 16 (pgvector) ← Redis 7 (queues + cache)
```
